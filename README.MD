## :information_source: Features

- [x] Mobile first/responsivity orientation development;
- [x] Admin panel to manage the site's content, like articles, project numbers, etc;
- [x] Dark/Light theme switch;
- [x] API to manage update and fetch requests with ease;
- [x] Advanced animation effects that run when scrolling the pages down;
- [x] Flickr api connection to get the most recent posted photos;
- [x] Validation of company's Excel spreadsheat's by checking user pc/username at a private mongodb database;

## :information_source: How to edit it

To clone this repository and make changes in the source code, you'll need [Git](https://git-scm.com) and  [Nodejs](https://nodejs.org/en/).

```bash
# Clone this repository
$ git clone https://github.com/lucasvtiradentes/biacaminha.com.br

# Go into the repository
$ cd biacaminha.com.br

# Install dependencies
$ npm install

# Run the code in development mode
$ npm run dev
```
